# 🗺️ Roadmap

OpenTutorAI is an open-source AI-powered learning platform that aims to democratize access to intelligent educational tools.  
Our mission is to empower learners and educators with adaptive feedback, personalized assistants, and collaborative learning spaces.

## 🛠️ Upcoming Milestone – v1.1.0 (Q3 2025)

- [ ] Avatar customization (name, tone, voice)
- [ ] Display tutor memory per session
- [ ] UI/UX enhancements (tooltips, cleaner layout)
- [ ] Engagement tracking dashboard (clicks, feedback, drop-off)
- [ ] Define and expose APIs for external extensions
