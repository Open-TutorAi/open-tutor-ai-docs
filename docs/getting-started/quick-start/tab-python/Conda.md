
# Install with Conda

1. **Create a Conda Environment:**

   ```bash
   conda create -n open-tutorai python=3.11
   ```

2. **Activate the Environment:**

   ```bash
   conda activate open-tutorai
   ```

3. **Install Open TutorAI:**

   ```bash
   pip install open-tutorai
   ```

4. **Start the Server:**

   ```bash
   open-tutorai serve
   ```
