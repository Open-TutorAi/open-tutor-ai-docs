
# Install with Conda

1. **Create a Conda Environment:**

   ```bash
   conda create -n open-tutor python=3.11
   ```

2. **Activate the Environment:**

   ```bash
   conda activate open-tutor
   ```

3. **Install Open TutorAI:**

   ```bash
   pip install open-tutor
   ```

4. **Start the Server:**

   ```bash
   open-webui serve
   ```
